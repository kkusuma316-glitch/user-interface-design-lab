CSS:
The word cascading means that a style applied to a parent element will also apply to all children elements within the parent.

Inline CSS:
An inline CSS is used to apply a unique style to a single HTML element. An inline CSS uses the style attribute of an HTML element.

Internal CSS:
An internal CSS is used to define a style for a single HTML page. An internal CSS is defined in the " section" of an HTML page, within a "style element".
